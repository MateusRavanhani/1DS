O kernel é o componente central de um sistema operativo, funcionando como um "cérebro" que gere o hardware e atua como uma ponte entre o software e os componentes físicos de um computador ou dispositivo. As suas funções incluem gerir recursos como a CPU, memória e periféricos, controlar processos, e garantir uma comunicação segura e eficiente entre o hardware e os programas, tornando-se assim uma camada de abstração essencial para o funcionamento do sistema. 





**Shell**

*O que é:*

Um shell é uma interface entre o utilizador e o kernel do sistema operacional, permitindo que o utilizador execute comandos para interagir com o sistema. 

Tipos:

Existem vários tipos de shells, como o sh (Bourne Shell) e o csh (C shell), mas o Bash é amplamente utilizado. 

*Função:*

Atua como um interpretador de comandos, executando as ações solicitadas pelo utilizador. 

 **Bash (Bourne-Again SHell)**

*O que é:*

Bash é um interpretador de comandos popular e um tipo de shell, criado como uma versão melhorada do Bourne Shell (sh). 

*Padrão:*

É o shell padrão na maioria das distribuições Linux, sendo usado interativamente ou para executar scripts de automação. 

*Vantagens:*

O Bash oferece funcionalidades avançadas que o sh não possui, como suporte a arrays, manipulação de strings mais rica e recursos de programação mais robustos. 

**Shell Scripts:**

Ao escrever um script de shell (um ficheiro de texto com comandos Bash), pode automatizar tarefas repetitivas, desde a organização de ficheiros até a instalação de software, tornando o trabalho mais eficiente.
